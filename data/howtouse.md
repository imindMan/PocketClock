# Requirements

<ul>
    <li>Python >= 3.0</li>
    <li>Tkinter module</li>
    <li>Windows (recommend >= 10) (actually you can use another OS, just read the document)</li>
    <li>Git and GitHub</li>
</ul>

Make sure these things have no annoying issues or errors

# Installation

First, make sure you passed all of the requirements <br>

Clone the respository: <br>
`$ git clone https://github.com/imindMan/PocketClock.git`

Move to the PocketClock folder, then run<br>
`$ python .\src\main.py`

## Troubles with OS

The requirements are the same, except the OS <br>
So, clone it, and before you run it, change somethng in the project.
So, the things to change are in the `PocketClock/src/main.py` <br>

<img src=".\tutorial1.png" alt="Screen sample" /> <br>

These are the shortcuts just work in Windows (and Linux I guess). So, if these shortcuts didn't work on your computer (e.g MacBook and iMac) then change these shortcuts to what you want

Then, it should work again, just do like above.

# Using the program

When you open this program, you should see this (not the day)
<img src=".\screen.png" alt="Screen sample">

We can click to the "Countdown" to move to the Countdown page, "Finish" to move to the Finish work page, and "Stopwatch" to move to the Stopwatch page <br>
Some page you have to use shortcut to move to, these are the shortcuts in this programs (or maybe your own):

<ul>
    <li>Ctrl + H : return to the Home page</li>
    <li>Ctrl + Shift + P: pomodoro page</li>
    <li>Shift + Tab : to-do list</li>
    <li>Ctrl + W: World time page</li>
    <li>Ctrl + R: reopen the program</li>
</ul>

Note: all the time in this program are military time, and the general format of the time is: <br>
`ddd dd-MMM-yy hh:mm:ss tt` <br>
Example: Mon 01-Jan-22 00:00:00 AM

## To-do list page

<img src=".\tutorial6.png" alt="Screen sample">

Reopen the program, now you will see your tasks appear on the screen. Click "Count" button at the bottom next to the plus button and just work!

To-do list page can just have 9 tasks only (I want to make it more but ...)

And if you want to change something in to-do list, like change the content or delete it, sorry but I didn't work with these features (lazy mode these days) so I guess you can open the `PocketClock\src\todo.txt` file to change something.
